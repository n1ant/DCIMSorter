## Описание

**DCIM Sorter** — десктопное приложение на Python для автоматической сортировки фотографий и видео по дате съёмки.

Программа предназначена для организации файлов (например, из папки `DCIM` смартфона) с автоматическим распределением по каталогам.

Определение даты выполняется:
- по имени файла (если содержит шаблон даты)
- по дате изменения файла (fallback)

---

## Основные возможности

### Интерфейс
- Графический интерфейс на **Tkinter**
- Выбор входной и выходной директории
- Окно настроек
- Отображение прогресса выполнения
- Блокировка интерфейса во время работы

### Работа с файлами
- Сортировка фото и видео по дате
- Поддержка распространённых форматов (JPG, PNG, HEIC, MP4 и др.)
- Обработка вложенных каталогов

### Режимы
- Иерархический: `Год/Месяц/День`
- Плоский: `YYYY-MM-DD`
- Копирование или перемещение файлов

### Надёжность
- Разрешение конфликтов имён файлов
- Перенос неподдерживаемых файлов в `_unsorted`
- Обработка ошибок без прерывания выполнения

## Запуск

### Основной запуск

Используйте один из файлов:

- `Run.vbs`
- `Run.bat`

Оба файла выполняют идентичный запуск программы.

**Примечание:**
- Windows: запуск через `.bat` или `.vbs`
- Linux / macOS: запуск вручную

```bash
python main.py
```

---

## Тестирование

Для запуска тестов используйте:

- `Run Tests.bat`

В проекте реализованы unit-тесты, покрывающие:
- логику сортировки
- обработку файлов
- вспомогательные функции
